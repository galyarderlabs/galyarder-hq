# Brainstorm Command

This command invokes the **brainstorming** skill to refine your ideas.

## What This Command Does

1. **Intent Exploration** - Asks clarifying questions to understand what you're REALLY building.
2. **Alternative Mapping** - Explores different architectural or UX paths.
3. **Requirement Locking** - Produces a structured design document.

---
**Note**: This command is powered by the `galyarder-framework:brainstorming` skill.
