# Galyarder Security Audit - Claude-Skills Benchmark

## Purpose
This note benchmarks Galyarder's Security department against the cloned `claude-skills` repository to identify high-value worker hires.

## Current Situation In Galyarder Security
### Current strengths
Galyarder already has a serious security identity:
- `security-guardian`
- `security-reviewer`
- `cyber-intel`
- `perseus`
- multiple red-team, threat-intel, and offensive-security skills

### Current weakness pattern
The likely weakness is not intent.
The likely weakness is missing structured worker depth in:
- cloud security
- security operations / incident command
- AI security
- compliance-adjacent security hygiene
