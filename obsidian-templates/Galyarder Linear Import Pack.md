# Galyarder Linear Import Pack

## Purpose
This document is a copy-friendly pack for manually creating the first Linear projects and issues.

It is based on the current digital company vision and the benchmark notes collected so far.

## Recommended First Projects
### Project 1
- Title: Obsidian Reporting System
- Team: Knowledge
- Goal: Make Obsidian the founder-facing memory and reporting layer.

Suggested issues:
- Define Founder Master Report Template
- Define Department Report Templates
- Define Decision Log Standard
- Define Milestone Summary Standard

### Project 2
- Title: Agent-to-Skill Mapping Rollout
- Team: Foundation
- Goal: Make every major agent explicitly backed by a mapped worker-skill stack.

Suggested issues:
- Finalize Founder Office Mapping
- Finalize Engineering Mapping
- Finalize Growth Mapping
- Finalize Security Mapping
- Finalize Legal-Finance Mapping
- Finalize Knowledge Mapping
- Identify Orphan Skills

### Project 3
- Title: Engineering Department Audit
- Team: Engineering
- Goal: Benchmark Galyarder Engineering against stronger external skill libraries and define hires.

Suggested issues:
- Audit Architecture Skills Against External Benchmarks
- Audit Code Review And Adversarial Review Gaps
- Audit Playwright And Test Automation Gaps
- Audit CI/CD And Observability Gaps
- Propose Engineering Import / Adapt / Replace Plan

### Project 4
- Title: Growth Department Audit
- Team: Growth
- Goal: Turn Growth into a real department with stronger worker depth and clearer reporting.

Suggested issues:
- Audit Current Growth Skills By Funnel Stage
- Audit External Marketing Libraries
- Define Growth Department Reporting Standard
- Identify Orphan Growth Skills

### Project 5
- Title: Founder Office Buildout
- Team: Founder Office
- Goal: Make the founder office a fully operating digital department.

Suggested issues:
- Finalize Founder Office Report Template
- Define Founder Office Metrics
- Map Fundraising Operator Outputs To Obsidian
- Define Founder Office Linear Workflow
- Audit Additional Founder / Capital Skill Hires
